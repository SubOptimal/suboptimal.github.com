Fork: extm3u.pl

Dieses Skript erzeugt eine Abspielliste im erweiterten M3U Format. Die Abspielliste kann zur Wiedergabe
mit Programmen wie z.B. Audacious, Totem, Winamp™ und vielen Anderen verwendet werden.
Die Ausgabe der Abspielliste erfolgt auf die Standardausgabe (STDOUT). Somit können andere Programmen sie
direkt einlesen.
Abspiellisten im erweiteten .m3u Format enthalten noch zusätzliche Informationen zu den einzelnen Stücken, die
Spieldauer und den Text der durch das Abspielprogramm angezeigt werden soll. Diese Informationen werden aus
den so genannten Tags der Musikdateien extrahiert.

Eigenschaften

* liest die Dateinamen von den angegebenen Verzeichnissen oder von der Standardeingabe (STDIN)
* Ausgabe der Abspielliste auf die Standardausgabe (STDOUT)
* unterstützte Media Tags: ID3 (MP3), Vorbis (FLAC, OGG) (optionale Perl Pakete benötigt)
* die Abspielliste kann in einer Zufallsreihenfolge erstellte werden
* Zeilenende (CR, CRLF) und Dateitrennzeichen ('/', '\') können definiert werden
* Open Source, verfügbar unter https://github.com/splitbrain/extm3u.git
