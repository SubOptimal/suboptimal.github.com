Fork: ctapi-cyberjack

Dieser Treiber für die Familie der REINER SCT cyberJack pinpad/e-com
USB Smart Card-Lesegeräte implementiert die CT-API 1.1 Schnittstelle
und ebenso die PC/SC Schnittstelle für pcsc-lite.

Folgende Reiner SCT Lesegeräte werden unterstützt:
ProduktVendorID:ProductID
REINER SCT cyberJack pinpad USB0c4d:0100
REINER SCT cyberJack pinpad USB0c4d:0100
REINER SCT cyberJack e-com USB0c4d:0100
REINER SCT cyberJack pinpad(a) USB0c4d:0300
REINER SCT cyberJack e-com(a)0c4d:0400
REINER SCT cyberJack e-com plus DUO0c4d:0400
REINER SCT cyberJack e-com plus BIO0c4d:0400
REINER SCT cyberJack e-com plus RFID0c4d:0400
REINER SCT cyberJack e-com plus0c4d:0400
REINER SCT cyberJack Secoder0c4d:0400
REINER SCT cyberJack e-com(f)0c4d:0401
REINER SCT cyberJack e-com BIO0c4d:0401

Informationen über die Smart Card Lesegeräte finden Sie auf der Herstellerseite
unter http://www.reiner-sct.com

Funktionen

* zu 100% im Userspace implementiert
* alle Zugriffe erfolgen über das USB devfs in /proc/bus/usb (oder /dev/bus/usb bei Verwendung von udev)
* ein Patch wurde implementiert, damit der Sourcecode mit gcc 4.7 kompiliert werden kann,
   inklusive einer Anleitung zum erstellung von Paketen für Debian Wheezy
* Open Source, verfügbar unter https://github.com/larskanis/ctapi-cyberjack.git

Fork des Quellcode-Repository zur Beteiligung: https://github.com/SubOptimal/ctapi-cyberjack.git
