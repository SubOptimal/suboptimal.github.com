Fork: ctapi-cyberjack

This driver for the REINER SCT cyberJack pinpad/e-com USB
family of smart card readers implements the CT-API 1.1 interface,
as well as the PC/SC interface of pcsc-lite.

The following Reiner-SCT readers are supported:
ProductVendorID:ProductID
REINER SCT cyberJack pinpad USB0c4d:0100
REINER SCT cyberJack pinpad USB0c4d:0100
REINER SCT cyberJack e-com USB0c4d:0100
REINER SCT cyberJack pinpad(a) USB0c4d:0300
REINER SCT cyberJack e-com(a)0c4d:0400
REINER SCT cyberJack e-com plus DUO0c4d:0400
REINER SCT cyberJack e-com plus BIO0c4d:0400
REINER SCT cyberJack e-com plus RFID0c4d:0400
REINER SCT cyberJack e-com plus0c4d:0400
REINER SCT cyberJack Secoder0c4d:0400
REINER SCT cyberJack e-com(f)0c4d:0401
REINER SCT cyberJack e-com BIO0c4d:0401

For more information about the smart card reader itself please
visit http://www.reiner-sct.com

Features

* implemented 100% in userspace
* all accesses are done via the usb devfs in /proc/bus/usb (or /dev/bus/usb for udev-based systems)
* patch implemented to compile with gcc 4.7, including a HOWTO for Debian Wheezy
* Open Source, available at https://github.com/larskanis/ctapi-cyberjack.git

source repository forked to contribute: https://github.com/SubOptimal/ctapi-cyberjack.git
