Java Bibliothek libevatr

libevatr ist eine Java Bibliothek, die den einfachen Zugriff auf den XML-RPC Service des Bundeszentralamt für Steuern (BZSt), zur Bestätigung von Umsatzsteuer-Identifikationsnummern (USt-IdNr) und Adressdaten europäischer Firmen, ermöglicht.

Funktionen

* einfache Bestätigung: Bestätigung der ausländischen USt-IdNr
* qualifizierte Bestätigung: Bestätigung der ausländischen USt-IdNr, Name der Firma, Ort des Firmensitzes
Mittels einer qualifizierten Bestätigung können auch PLZ, Straße und Hausnummer des Firmensitzes validiert werden.
Zusätzlich kann eine amtliche Bestätigungsmitteilung angefordert werden.
Achtung: Diese wird vom Bundeszentralamt für Steuern (BZSt) auf postalischem Weg an die anfragende Firma versandt.

Offizielle Webseite der XML-RPC Schnittstelle beim Bundeszentralamt für Steuern (BZSt):
http://evatr.bff-online.de/eVatR/xmlrpc
